######WHEN A PROJECT IS NEW
On github.com create a repository
a. provide a repository name
b. Set your privacy
c. Get your repo url on creation

- start processes from the git terminal
1.Initialize the git folder : git init
2. connect the local project to the repo url : git rempte add origin <url>
3.Initial adding of files : git add .
4. make commit : git commit -m "write a commit message"
5. Ensure you have a password token to allow push
6. push using token via this command
git push origin master
where master is referencing the default branch where work will be saved
